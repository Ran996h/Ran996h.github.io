# HTML 小游戏：猜数字 以下是一个简单的猜数字游戏的HTML代码。游戏会随机生成1-100之间的数字，玩家需要猜测这个数字。 ```html 猜数字游戏

猜数字游戏

我已经想好了一个1到100之间的数字，你能猜出来吗？

尝试次数: 0
``` ## 游戏说明 1. 游戏会随机生成1-100之间的一个数字 2. 玩家在输入框中输入猜测的数字 3. 游戏会提示猜测的数字是"太大"还是"太小" 4. 玩家需要根据提示继续猜测，直到猜中为止 5. 游戏会记录尝试次数 6. 猜中后可以点击"重新开始"按钮开始新游戏 ## 如何使用 1. 将上面的代码复制到一个文本编辑器中 2. 保存为 `.html` 文件（例如 `number-guessing-game.html`） 3. 在浏览器中打开该文件即可开始游戏 您可以根据需要修改样式或游戏规则，比如改变数字范围、添加难度级别等。
