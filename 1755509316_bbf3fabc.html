# HTML 小游戏：猜数字 下面是一个简单的猜数字HTML小游戏代码。玩家需要在1-100之间猜一个随机生成的数字。 ```html 猜数字游戏

猜数字游戏

我已经想好了一个1到100之间的数字，你能猜出来吗？

尝试次数: 0
``` ## 游戏说明 1. 游戏会随机生成一个1-100之间的数字 2. 玩家在输入框中输入猜测的数字 3. 游戏会提示猜测是"太大"还是"太小" 4. 猜中后显示恭喜信息并统计尝试次数 5. 可以点击"再玩一次"按钮重新开始游戏 ## 如何使用 1. 将上面的代码复制到一个文本编辑器中 2. 保存为`.html`文件（例如`guess-number.html`） 3. 在浏览器中打开该文件即可开始游戏 您可以根据需要修改样式、数字范围或添加更多功能来扩展这个小游戏。
