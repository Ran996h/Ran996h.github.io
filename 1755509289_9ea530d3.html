# HTML 小游戏：猜数字 下面是一个简单的猜数字HTML小游戏代码。玩家需要在1-100之间猜一个随机生成的数字。 ```html 猜数字游戏

猜数字游戏

我已经想好了一个1-100之间的数字，你能猜出来吗？

尝试次数: 0

猜测历史:

    ``` ## 游戏说明 1. 游戏会随机生成一个1-100之间的数字 2. 玩家在输入框中输入猜测的数字 3. 点击"猜一猜"按钮或按回车键提交猜测 4. 游戏会提示猜测是"太大"还是"太小" 5. 所有猜测历史会显示在下方 6. 猜中后游戏会显示恭喜信息并禁用输入框 7. 点击"重新开始"按钮可以开始新游戏 ## 如何运行 1. 将上面的代码复制到一个文本编辑器中 2. 保存为 `.html` 文件（例如 `guess-number.html`） 3. 在浏览器中打开该文件即可开始游戏 您可以根据需要修改样式或游戏规则，比如改变数字范围、添加难度级别等。
